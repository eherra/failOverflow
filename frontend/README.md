### Getting started

1. Add .env file. See example:
[frontend env](https://github.com/eherra/failOverflow/blob/main/frontend/.env.example)

2. Run command 
```
npm install && npm start
```

Frontend is running at http://localhost:3000


### Other commands

Linting
```
npm run lint
```

Formatting
```
npm run format
```
